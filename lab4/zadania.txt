1) Wypisz komendy linii poleceń umożliwiające:

wyświetlenie informacji o poleceniach (pomocy)    help
wyświetlanie zawartości bieżącego katalogu        ls    
utworzenie katalogu o nazwie 'katalog'            mkdir c:\katalog
utworzenia pliku o nazwie 'plik.txt'              touch plik.txt
wyświetlenie zawartości pliku 'plik.txt'          cat plik.txt
przejście do katalogu nadrzędnego                 cd ..
przejście do katalogu 'lab4'                      cd lab4
zmianę nazwy pliku 'plik.txt' na 'inny_plik.txt'  mv plik.txt inny_plik.txt  

2) Wypróbuj na własnym repozytorium oraz podaj komendy gita służącą do:

podglądu wszystkich zatwierdzonych zmian                              git log  
podglądu wprowadzonych w repozytorium zmian                           git diff  
podglądu wprowadzonych zmian plików które są juz w przechowalni       git diff-staged 
podglądu zmian wprowadzonych w danym pliku                            git status
usuwania plików z repozytorium                                        git rm  
dodawanie plików do przechowalni                                      git add
usuwanie plików z przechowalni                                        git reset
przenoszenia plików znajdujących się w przechowalni do innej gałęzi   git stash  
usuwania gałęzi                                                       git branch-d
